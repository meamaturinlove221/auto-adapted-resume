# auto-adapted-resume
A small, evidence-first resume adaptation skill.

The project provides a reusable `SKILL.md` workflow for turning a target job description into:

- a JD parsing summary
- a JD-resume alignment table
- a tailored resume draft
- an interview-prep pack
- a claim-source map

It is designed for people who already have experience but struggle to decide which parts of the resume should change for each role.

## Why this exists

Most resume prompts start with: "Make my resume better."

This skill starts with: "What does the JD require, and where is the evidence in the resume?"

## What it does not do

- It does not fabricate experience.
- It does not auto-apply to jobs.
- It does not guarantee interview results.
- It does not replace human review.
